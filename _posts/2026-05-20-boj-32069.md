## 풀이

```cpp
#include <iostream>
#include <queue>
#include <set>

using namespace std;

long long l, a;
int n, k;
const long long d[] = {-1, 1};
queue<pair<long long, long long>> q;
set<long long> v;

bool in(long long x) {
    return 0 <= x && x <= l;
}

int main() {
    cin >> l >> n >> k;
    for (int i = 0; i < n; ++i) {
        cin >> a;
        q.push({a, 0});
        v.insert(a);
    }

    while (!q.empty()) {
        auto [x, d0] = q.front(); q.pop();
        cout << d0 << '\n';
        if(--k == 0)  return 0;
        for (int i = 0; i < 2; ++i) {
            long long nx = x + d[i];
            if (in(nx) && !v.count(nx)) {
                q.push({nx, d0 + 1});
                v.insert(nx);
            }
        }
    }
}

```

## 해설

### 핵심 아이디어
가로등이 없는 구간을 최소 수의 가로등으로 커버. 그리디 + 자료구조(set) 활용.

### 풀이 과정
- 가로등 위치를 정렬.
- 커버되지 않는 구간을 set에 저장하며 최소 개수를 선택하는 그리디.

### 시간복잡도
O(N log N)
