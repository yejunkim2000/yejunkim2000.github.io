## 문제 정보

- 번호: 10814
- 난이도: 실버 5
- 링크: [https://www.acmicpc.net/problem/10814](https://www.acmicpc.net/problem/10814)

## 문제 요약

회원들을 나이 오름차순으로 정렬하되,
나이가 같으면 먼저 가입한 순서(입력 순서)를 유지한다.

## 풀이

나이가 같을 때 입력 순서를 유지해야 하므로 안정 정렬이 필요하다.
C++의 `stable_sort()`를 사용하면 같은 키의 원소는 원래 순서를 유지한다.
또는 pair에 인덱스를 같이 저장해서 comparator에서 처리해도 된다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;

    vector<pair<int, string>> members(n);
    for (int i = 0; i < n; i++) {
        cin >> members[i].first >> members[i].second;
    }

    stable_sort(members.begin(), members.end(), [](const auto& a, const auto& b) {
        return a.first < b.first;
    });

    for (const auto& m : members) {
        cout << m.first << ' ' << m.second << '\n';
    }

    return 0;
}
```

## 배운 점

C++ `sort()`는 불안정 정렬이고 `stable_sort()`는 안정 정렬이다.
같은 키를 가진 원소의 순서를 보장해야 할 때는 반드시 `stable_sort()`를 써야 한다.
