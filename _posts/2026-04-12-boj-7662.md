## 문제 정보

- 번호: 7662
- 난이도: 골드 4
- 링크: [https://www.acmicpc.net/problem/7662](https://www.acmicpc.net/problem/7662)

## 문제 요약

이중 우선순위 큐(Double-ended Priority Queue)를 구현한다.
두 가지 연산을 처리한다.

- `I n`: 정수 n을 큐에 삽입
- `D 1`: 최댓값 삭제
- `D -1`: 최솟값 삭제

모든 연산을 처리한 후 큐가 비어있으면 `EMPTY`, 아니면 최댓값과 최솟값을 출력한다.

---

## 핵심 아이디어

최댓값과 최솟값을 동시에 빠르게 삭제해야 하므로 일반 우선순위 큐 하나로는 부족하다.

**map을 사용한다.**

`map<int, int>`에서 key는 숫자 값, value는 해당 숫자의 개수를 저장한다.

- 삽입: `mp[n]++`
- 최댓값 삭제: `mp.rbegin()` (map의 마지막 원소 = 최댓값)
- 최솟값 삭제: `mp.begin()` (map의 첫 원소 = 최솟값)
- 개수가 0이 되면 map에서 제거

map은 key를 자동으로 정렬하기 때문에 항상 최솟값은 begin(), 최댓값은 rbegin()으로 O(1)에 접근 가능하다.
삽입, 삭제는 O(log N).

---

## 풀이 과정

**1단계 - 삽입 처리**

`mp[n]++`으로 숫자 n의 개수를 증가시킨다.

**2단계 - 최댓값 삭제**

`auto it = mp.end(); --it;`로 마지막 원소를 가져온다.
`it->second`를 1 감소. 0이 되면 `mp.erase(it)`.

**3단계 - 최솟값 삭제**

`auto it = mp.begin()`으로 첫 원소를 가져온다.
동일하게 처리.

**4단계 - 큐가 비어있는지 확인**

삭제 연산 전에 `mp.empty()` 체크. 비어있으면 아무것도 안 한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int T;
    cin >> T;

    while (T--) {
        int K;
        cin >> K;

        map<int, int> mp;

        while (K--) {
            char op;
            int n;
            cin >> op >> n;

            if (op == 'I') {
                mp[n]++;
            } else {
                if (mp.empty()) continue;

                if (n == 1) {
                    auto it = mp.end();
                    --it;
                    it->second--;
                    if (it->second == 0) mp.erase(it);
                } else {
                    auto it = mp.begin();
                    it->second--;
                    if (it->second == 0) mp.erase(it);
                }
            }
        }

        if (mp.empty()) {
            cout << "EMPTY\n";
        } else {
            auto maxIt = mp.end(); --maxIt;
            cout << maxIt->first << ' ' << mp.begin()->first << '\n';
        }
    }

    return 0;
}
```

## 틀렸던 포인트 / 주의사항

- 삭제 연산 시 큐가 비어있는 경우를 반드시 처리해야 한다. 빈 map에서 begin/end 접근하면 UB.
- 같은 숫자가 여러 개 있을 수 있으므로 개수를 관리해야 한다. erase를 바로 쓰면 안 된다.
- `mp.end()`는 마지막 원소의 다음을 가리키므로 `--it`으로 마지막 원소를 가져와야 한다.

---

## 배운 점

`map`은 key가 자동 정렬되므로 **최솟값과 최댓값을 동시에 관리**하는 자료구조로 활용 가능하다.
이중 우선순위 큐를 구현하는 여러 방법(힙 두 개 + lazy deletion, 세그먼트 트리 등) 중
map을 활용하는 방법이 코드가 가장 간결하고 실수도 적다.
