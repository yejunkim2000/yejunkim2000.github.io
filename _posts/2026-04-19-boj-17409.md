## 문제

크기가 N인 수열 A와 정수 K가 주어졌을 때, A의 증가하는 부분 수열 중에서 길이가 K인 것의 개수를 구해보자.

## 입력

첫째 줄에 N, K가 주어진다. 둘째 줄에 수열 A1, A2, ..., AN이 주어진다.

## 출력

첫째 줄에 A의 증가하는 부분 수열 중에서 길이가 K인 것의 개수를 109+7로 나눈 나머지를 출력한다.

## 풀이

```cpp
#include <stdio.h>

#define MOD 1000000007

long long seg[11][262145];

void u(int l, int r, int qI, long long qV, int k) {
    int sI = 1;
    while (l < r) {
        seg[k][sI] += qV;
        seg[k][sI] %= MOD;
        int m = (l + r) / 2;
        if (qI <= m) {
            r = m;
            sI = sI * 2;
        } else {
            l = m + 1;
            sI = sI * 2 + 1;
        }
    }
    seg[k][sI] += qV;
    seg[k][sI] %= MOD;
}

long long rs(int l, int r, int qL, int qR, int sI, int k) {
    if (qR < l || r < qL) return 0;
    if (qL <= l && r <= qR) return seg[k][sI];
    return (rs(l, (l + r) / 2, qL, qR, sI * 2, k) + rs((l + r) / 2 + 1, r, qL, qR, sI * 2 + 1, k)) % MOD;
}

int main() {
    int n, k;
    scanf("%d %d", &n, &k);
    int nn = n;
    while (nn--) {
        int x;
        scanf("%d", &x);
        for (int i = 10; i > 1; i--) {
            long long cnt = rs(1, n, 1, x, 1, i - 1);
            u(1, n, x, cnt, i);
        }
        u(1, n, x, 1, 1);
    }

    printf("%lld\n", seg[k][1]);
    return 0;
}

```

## 해설

### 핵심 아이디어
길이 k인 증가 부분 수열의 개수를 세는 DP + 세그먼트 트리.

### 풀이 과정
- `dp[k][i]` = A[i]로 끝나는 길이 k의 증가 부분 수열 수.
- A[i]보다 작은 값에서의 dp[k-1] 합 = 세그먼트 트리 구간 합 쿼리.
- K개의 세그먼트 트리 유지.

### 시간복잡도
O(N × K × log N)
