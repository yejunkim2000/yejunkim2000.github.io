## 문제 정보

- 번호: 10989
- 난이도: 브론즈 1
- 링크: [https://www.acmicpc.net/problem/10989](https://www.acmicpc.net/problem/10989)

## 문제 요약

N개의 수를 오름차순으로 정렬해서 출력한다.
N은 최대 10,000,000이고 메모리 제한이 8MB로 매우 작다.
수의 범위는 1 이상 10,000 이하다.

## 풀이

메모리 제한이 8MB라 vector에 전부 담으면 초과한다.
수의 범위가 1~10,000으로 제한되어 있으므로 카운팅 정렬을 사용한다.
각 숫자가 몇 번 등장했는지 count 배열에 기록한 뒤 순서대로 출력한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;

    int count[10001] = {};
    for (int i = 0; i < n; i++) {
        int x;
        cin >> x;
        count[x]++;
    }

    for (int i = 1; i <= 10000; i++) {
        for (int j = 0; j < count[i]; j++) {
            cout << i << '\n';
        }
    }

    return 0;
}
```

## 배운 점

메모리 제한이 빡빡할 때는 카운팅 정렬을 고려해야 한다.
수의 범위가 제한적일 때만 사용 가능하지만 O(N)으로 매우 빠르다.
stack 배열(`int count[10001]`)은 heap(`vector`)보다 메모리를 훨씬 적게 쓴다.
