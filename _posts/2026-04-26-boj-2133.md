## 문제

3&times;N 크기의 벽을 2&times;1, 1&times;2 크기의 타일로 채우는 경우의 수를 구해보자.

## 입력

첫째 줄에 N(1 &le; N &le; 30)이 주어진다.

## 출력

첫째 줄에 경우의 수를 출력한다.

## 예제

### 예제 입력 1
```
2
```

### 예제 출력 1
```
3
```

---

## 풀이

```cpp
#include <iostream>
using namespace std;

int main()
{
    int N;
    cin >> N;
    int d[31];
    d[0] = 1;
    d[1] = 0;
    d[2] = 3;

    for (int n = 3; n <= N; n++)
    {
        if (n % 2 != 0) d[n] = 0;
        else
        {
            for (int i = 2; i <= N; i += 2)
            {
                if (i == 2) d[n] = d[n - i] * d[2];
                else if((n-i) >= 0) d[n] += d[n - i] * 2;
            }
        }
    }
    cout <<d[N];
}
```


---

## 해설

### 핵심 아이디어
3×N 타일 채우기 DP. 점화식 유도가 핵심.

### 풀이 과정
- N이 홀수이면 불가능(0).
- `dp[N] = 3 * dp[N-2] + 2 * (dp[N-4] + dp[N-6] + ... + dp[0])`
- 짝수 N에 대해서만 계산하며 이전 짝수 dp 값들을 누적.

### 시간복잡도
O(N)
