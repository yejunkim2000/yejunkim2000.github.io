## 문제 정보

- 번호: 25305
- 난이도: 브론즈 2
- 링크: [https://www.acmicpc.net/problem/25305](https://www.acmicpc.net/problem/25305)

## 문제 요약

N명의 점수 중 상위 k명이 상을 받는다.
상을 받는 최저 점수(커트라인)를 출력한다.

## 풀이

점수를 내림차순으로 정렬한 뒤 k번째 값을 출력한다.
인덱스는 0부터 시작하므로 k번째는 `arr[k-1]`이다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int n, k;
    cin >> n >> k;

    vector<int> scores(n);
    for (int i = 0; i < n; i++) cin >> scores[i];

    sort(scores.begin(), scores.end(), greater<int>());

    cout << scores[k - 1] << '\n';

    return 0;
}
```

## 배운 점

정렬 후 인덱스로 접근하는 가장 기본적인 패턴이다.
`greater<int>()`로 내림차순 정렬을 간단하게 구현할 수 있다.
k번째를 구할 때 인덱스가 k-1임을 항상 주의해야 한다.
