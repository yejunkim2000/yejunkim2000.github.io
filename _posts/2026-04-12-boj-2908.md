## 문제 정보

- 번호: 2908
- 난이도: 브론즈 2
- 링크: [https://www.acmicpc.net/problem/2908](https://www.acmicpc.net/problem/2908)

## 문제 요약

세 자리 수 두 개를 거꾸로 읽은 뒤 더 큰 수를 출력한다.

## 풀이

문자열로 입력받아 reverse()로 뒤집은 뒤 정수로 변환해서 비교한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string a, b;
    cin >> a >> b;

    reverse(a.begin(), a.end());
    reverse(b.begin(), b.end());

    cout << max(stoi(a), stoi(b)) << '\n';

    return 0;
}
```

## 배운 점

`reverse()`로 문자열을 뒤집고 `stoi()`로 정수 변환하는 패턴은 자주 쓰인다.
간단한 문제일수록 코드를 짧고 명확하게 쓰는 연습이 필요하다.
