## 문제

N-Queen 문제는 크기가 N &times; N인 체스판 위에 퀸 N개를 서로 공격할 수 없게 놓는 문제이다.

N이 주어졌을 때, 퀸을 놓는 방법의 수를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 N이 주어진다. (1 &le; N

## 출력

첫째 줄에 퀸 N개를 서로 공격할 수 없게 놓는 경우의 수를 출력한다.

## 예제

### 예제 입력 1
```
8
```

### 예제 출력 1
```
92
```

---

## 풀이

```cpp
#include <iostream>
using namespace std;

int arr[15];
int N, total = 0;

bool check(int n)
{
  for(int i = 0; i < n; i++)
    if(arr[i] == arr[n] || abs(arr[n] - arr[i]) == n - i)
      return false;
  return true;
  
}

void nqueen(int x)
{
  if(x == N)
  {
    total++;
  }
  else
  {
    for(int i = 0; i< N; i++)
    {
      arr[x] = i;
      if(check(x))
      {
        nqueen(x+1);
      }
    }
  }
}
int main()
{
  ios_base::sync_with_stdio(false);
  cin.tie(NULL);
  cout.tie(NULL);
  cin >> N;
  nqueen(0);
  cout << total;
}
```

## 해설

### 핵심 아이디어
N-Queen: N×N 체스판에 N개의 퀸을 서로 공격하지 않게 배치하는 경우의 수. 백트래킹.

### 풀이 과정
1. 행 단위로 퀸 배치.
2. 각 열, 대각선에 퀸 있는지 체크.
3. 가능하면 다음 행으로, 불가능하면 백트래킹.

### 시간복잡도
O(N!) — 실제론 가지치기로 훨씬 빠름
