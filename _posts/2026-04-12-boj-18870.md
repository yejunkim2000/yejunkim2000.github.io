## 문제 정보

- 번호: 18870
- 난이도: 실버 2
- 링크: [https://www.acmicpc.net/problem/18870](https://www.acmicpc.net/problem/18870)

## 문제 요약

N개의 수가 주어질 때, 각 수를 좌표 압축한 결과를 출력한다.
좌표 압축이란 각 수를 "자신보다 작은 수가 몇 개인가"로 변환하는 것이다.

## 풀이

중복 제거 후 정렬한 배열을 만든다.
`lower_bound()`로 원래 배열의 각 원소가 압축 배열에서 몇 번째 인덱스인지 찾는다.

예를 들어 `[2, 4, -10, 4, -9]`가 있으면
정렬 + 중복 제거: `[-10, -9, 2, 4]`
`lower_bound(2)` → 인덱스 2
`lower_bound(4)` → 인덱스 3
결과: `[2, 3, 0, 3, 1]`

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;

    vector<int> arr(n);
    for (int i = 0; i < n; i++) cin >> arr[i];

    vector<int> sorted_arr = arr;
    sort(sorted_arr.begin(), sorted_arr.end());
    sorted_arr.erase(unique(sorted_arr.begin(), sorted_arr.end()), sorted_arr.end());

    for (int i = 0; i < n; i++) {
        cout << lower_bound(sorted_arr.begin(), sorted_arr.end(), arr[i]) - sorted_arr.begin();
        if (i < n - 1) cout << ' ';
    }
    cout << '\n';

    return 0;
}
```

## 배운 점

좌표 압축은 값의 범위가 크지만 상대적 순서만 중요할 때 사용하는 기법이다.
`lower_bound()`는 정렬된 배열에서 특정 값의 위치를 O(log N)에 찾아준다.
이터레이터 빼기 연산으로 인덱스를 바로 구할 수 있다.
실제로 세그먼트 트리, BIT 등 고급 자료구조의 전처리로 자주 쓰인다.
