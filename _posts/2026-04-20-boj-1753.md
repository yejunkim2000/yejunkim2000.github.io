## 문제

방향그래프가 주어지면 주어진 시작점에서 다른 모든 정점으로의 최단 경로를 구하는 프로그램을 작성하시오. 단, 모든 간선의 가중치는 10 이하의 자연수이다.

## 입력

첫째 줄에 정점의 개수 V와 간선의 개수 E가 주어진다. (1 &le; V &le; 20,000, 1 &le; E &le; 300,000) 모든 정점에는 1부터 V까지 번호가 매겨져 있다고 가정한다. 둘째 줄에는 시작 정점의 번호 K(1 &le; K &le; V)가 주어진다. 셋째 줄부터 E개의 줄에 걸쳐 각 간선을 나타내는 세 개의 정수 (u, v, w)가 순서대로 주어진다. 이는 u에서 v로 가는 가중치 w인 간선이 존재한다는 뜻이다. u와 v는 서로 다르며 w는 10 이하의 자연수이다. 서로 다른 두 정점 사이에 여러 개의 간선이 존재할 수도 있음에 유의한다.

## 출력

첫째 줄부터 V개의 줄에 걸쳐, i번째 줄에 i번 정점으로의 최단 경로의 경로값을 출력한다. 시작점 자신은 0으로 출력하고, 경로가 존재하지 않는 경우에는 INF를 출력하면 된다.

## 예제

### 예제 입력 1
```
5 6
1
5 1 1
1 2 2
1 3 3
2 3 4
2 4 5
3 4 6
```

### 예제 출력 1

## 풀이

```cpp
#include<iostream>
#include<queue>
#include<vector>
#define endl "\n"
#define MAX 20010
#define INF 987654321

using namespace std;

int V, E, S;
int dist[MAX];
vector<pair<int, int>> adj[MAX];

void input() {
    cin >> V >> E >> S;
    for (int i = 0; i < E; i++) {
        int a, b, c;
        cin >> a >> b >> c;
        adj[a].push_back({b, c});
    }
    fill(dist, dist + V + 1, INF);
}

void dijkstra() {
    priority_queue<pair<int, int>> pq;
    pq.push({0, S});
    dist[S] = 0;
    
    while (!pq.empty()) {
        int cost = -pq.top().first;
        int cur = pq.top().second;
        pq.pop();

        for (auto& next : adj[cur]) {
            int nxt = next.first;
            int nCost = next.second;
            if (dist[nxt] > cost + nCost) {
                dist[nxt] = cost + nCost;
                pq.push({-dist[nxt], nxt});
            }
        }
    }
    
    for (int i = 1; i <= V; i++) {
        if (dist[i] == INF) cout << "INF" << endl;
        else cout << dist[i] << endl;
    }
}

void solve() {
    input();
    dijkstra();
}

int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    
    solve();
    return 0;
}

```

## 해설

### 핵심 아이디어
다익스트라 알고리즘으로 단일 출발점 최단 경로 계산.

### 풀이 과정
1. 출발 노드를 우선순위 큐(min-heap)에 넣고 시작.
2. 가장 짧은 거리의 노드를 꺼내 인접 노드 거리 갱신.
3. 이미 방문한 노드는 skip.
4. 도달 불가 노드는 INF 출력.

### 시간복잡도
O((V + E) log V) — V: 노드 수, E: 간선 수
