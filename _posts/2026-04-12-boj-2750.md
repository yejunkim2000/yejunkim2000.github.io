## 문제 정보

- 번호: 2750
- 난이도: 브론즈 2
- 링크: [https://www.acmicpc.net/problem/2750](https://www.acmicpc.net/problem/2750)

## 문제 요약

N개의 수를 오름차순으로 정렬해서 출력한다.
N은 최대 1,000이고 수는 중복되지 않는다.

## 풀이

N이 최대 1,000으로 작기 때문에 어떤 정렬을 써도 통과한다.
STL의 sort()를 사용하면 O(N log N)으로 간단하게 해결된다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;

    vector<int> arr(n);
    for (int i = 0; i < n; i++) cin >> arr[i];

    sort(arr.begin(), arr.end());

    for (int x : arr) cout << x << '\n';

    return 0;
}
```

## 배운 점

가장 기본적인 정렬 문제.
N이 작아서 어떤 정렬이든 통과하지만 STL sort()를 습관적으로 쓰는 게 좋다.
