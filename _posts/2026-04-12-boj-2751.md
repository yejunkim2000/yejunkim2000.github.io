## 문제 정보

- 번호: 2751
- 난이도: 실버 5
- 링크: [https://www.acmicpc.net/problem/2751](https://www.acmicpc.net/problem/2751)

## 문제 요약

N개의 수를 오름차순으로 정렬해서 출력한다.
N은 최대 1,000,000이다.

## 풀이

N이 최대 100만이라 O(N²) 정렬은 시간 초과다.
STL sort()는 O(N log N)이므로 통과한다.
입출력이 많으므로 `ios::sync_with_stdio(false)`와 `cin.tie(NULL)`을 반드시 설정한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;

    vector<int> arr(n);
    for (int i = 0; i < n; i++) cin >> arr[i];

    sort(arr.begin(), arr.end());

    for (int x : arr) cout << x << '\n';

    return 0;
}
```

## 배운 점

N이 커지면 입출력 속도가 중요하다.
`sync_with_stdio(false)` + `cin.tie(NULL)` 조합은 C++ 입출력 최적화의 기본이다.
