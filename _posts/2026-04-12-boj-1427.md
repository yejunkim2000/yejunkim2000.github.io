## 문제 정보

- 번호: 1427
- 난이도: 브론즈 2
- 링크: [https://www.acmicpc.net/problem/1427](https://www.acmicpc.net/problem/1427)

## 문제 요약

주어진 수의 각 자리를 내림차순으로 정렬해서 출력한다.

## 풀이

수를 문자열로 받아서 각 문자를 내림차순 정렬한 뒤 출력한다.
`greater<char>()`를 comparator로 사용하면 내림차순이 된다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string n;
    cin >> n;

    sort(n.begin(), n.end(), greater<char>());

    cout << n << '\n';

    return 0;
}
```

## 배운 점

수를 정수로 받지 않고 문자열로 받으면 각 자리를 바로 정렬할 수 있다.
한 자리 숫자는 문자 비교와 크기 비교가 동일하므로 char 정렬이 그대로 적용된다.
`greater<char>()`로 내림차순을 간단하게 구현할 수 있다.
