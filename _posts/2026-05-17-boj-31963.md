## 문제

길이 $N$인 양의 정수열 $A_1, \dots , A_N$이 주어진다. 이 수열을 오름차순으로 만들려 한다. 수열 $A_1, \dots , A_N$이 오름차순이라는 것은, 각 $i$ ($1 &le; i &le; N - 1$)에 대해 $A_i &le; A_{i+1}$이라는 것이다.

수열 $A$를 오름차순으로 만들기 위해, 수열 $A$에 다음 연산을 몇 번이든 반복해서 적용할 수 있다.


	어떤 $i$ ($1 &le; i &le; N$)에 대해 $A_i$에 $2$를 곱한다.


연산을 최소 횟수로 적용해서 $A$를 오름차순으로 만들고 싶다. 이때, 최소 횟수를 구하라.

## 입력

첫 번째 줄에 $N$이 주어진다.

두 번째 줄에 $A_1, \dots , A_N$이 주어진다.

## 출력

첫 번째 줄에 답을 출력한다.

## 예제

### 예제 입력 1
```
5
3 1 4 1 5
```

### 예제 출력 1
```
4
```

### 예제 입력 2
```
5
3 1 5 1 5
```

### 예제 출력 2
```
6
```

### 예제 입력 3
```
5
1 2 3 4 5
```

### 예제 출력 3
```
0
```

## 풀이

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int N;
vector<int> inp(250010);

long long diff, bef, cur;
long long ret;

vector<long long> check(1000010);

int getDiff(int a, int b, int f) {
    int ret = 0;
    
    while (a <= b) {
        if (f && a == b) break;
        
        a *= 2;
        ret++;
    }
    return ret;
}

int main() {
    cin >> N;
    
    for (int i = 1; i <= N; i++) {
        cin >> inp[i];
    }
    
    for (int i = 2; i <= N; i++) {
        if (inp[i - 1] < inp[i]) {
            diff = getDiff(inp[i - 1], inp[i], 0);
            
            bef = check[inp[i - 1]] - diff;
            if (bef >= check[inp[i]]) {
                check[inp[i]] = bef + 1;
            }
            
        } else if (inp[i - 1] > inp[i]) {
            diff = getDiff(inp[i], inp[i - 1], 1);
            
            cur = check[inp[i]] - diff;
            if (check[inp[i - 1]] > cur) {
                check[inp[i]] = check[inp[i - 1]] + diff;
            }
        }
        
        ret += check[inp[i]];
    }
    
    cout << ret;
    return 0;
}

```

## 해설

### 핵심 아이디어
배열에서 원소를 2배로 만드는 연산으로 모두 같게 만드는 최소 연산 수. 수학적 분석.

### 풀이 과정
- 최종 값은 항상 최댓값의 2의 배수 중 하나.
- 각 원소를 목표값에 맞추기 위한 연산 횟수 합산.
- 그리디로 최적 목표값 선택.

### 시간복잡도
O(N log(max))
