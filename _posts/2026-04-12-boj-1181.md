## 문제 정보

- 번호: 1181
- 난이도: 실버 5
- 링크: [https://www.acmicpc.net/problem/1181](https://www.acmicpc.net/problem/1181)

## 문제 요약

N개의 단어를 길이 오름차순, 길이가 같으면 사전순으로 정렬한다.
중복 단어는 하나만 출력한다.

## 풀이

sort()의 comparator에 길이 우선, 사전순 차순 조건을 작성한다.
중복 제거는 sort 후 unique() + erase()로 처리한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;

    vector<string> words(n);
    for (int i = 0; i < n; i++) cin >> words[i];

    sort(words.begin(), words.end(), [](const string& a, const string& b) {
        if (a.size() != b.size()) return a.size() < b.size();
        return a < b;
    });

    words.erase(unique(words.begin(), words.end()), words.end());

    for (const string& w : words) cout << w << '\n';

    return 0;
}
```

## 배운 점

C++ sort()의 comparator로 다중 조건 정렬을 구현할 수 있다.
`unique()`는 인접한 중복만 제거하므로 반드시 sort() 후에 써야 한다.
`unique()`가 반환하는 이터레이터 이후 원소를 erase()로 제거해야 완전히 삭제된다.
